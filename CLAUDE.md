## 버전 시스템 (X.X.X.X)

| 자리 | 의미 | 예시 |
|------|------|------|
| 첫째 | 시스템 전면 교체 (아키텍처/플랫폼 변경) | 1.x.x.x → 2.x.x.x |
| 둘째 | 대규모 기능 업데이트 | x.1.x.x → x.2.x.x |
| 셋째 | 소규모 기능 추가 | x.x.1.x → x.x.2.x |
| 넷째 | 버그 수정 / 오류 수정 | x.x.x.1 → x.x.x.2 |

- **업데이트(3·2번째 자리)** = 기능 추가 위주
- **수정(4번째 자리)** = 기능 변경 없는 버그픽스

## 파일명
