체납대장 분석
워크스테이션

엑셀 파일을 열고, 필터하고, 비교하고, 저장하는 모든 기능을 이 페이지에서 직접 체험해보세요. 기능을 이해한 뒤 실제 앱을 훨씬 빠르게 활용할 수 있습니다.

Quick Start

빠른 시작 — 3단계

처음 사용하시는 분을 위한 기본 플로우입니다.

📂
① 파일 열기
좌측 탐색기에서 폴더를 선택하거나 엑셀 파일을 드래그&드롭 합니다.
지원: .xlsx · .xls · .xlsm · .csv
🔍
② 검색 & 필터
통합 검색창에 키워드를 입력하거나 각 열 아래 필터 입력란을 사용합니다.
퍼지 검색 · 연산자 필터 · 스마트 필터
💾
③ 저장
셀 편집 후 '덮어쓰기' 또는 '새 이름으로 저장' 버튼을 클릭합니다.
원본 서식 완전 보존 · Undo 지원
💡
탐색기 vs 드래그&드롭: 탐색기에서 파일을 열면 덮어쓰기가 가능합니다. 드래그&드롭으로 열면 브라우저 보안 정책상 새 파일로만 저장됩니다.

Special Feature

📥 VLOOKUP+ (데이터 입력)

두 파일을 나란히 놓고 공통 키(이름·번호 등)로 연결해 오른쪽 파일의 값을 왼쪽 파일에 자동으로 채웁니다. 엑셀 VLOOKUP 함수와 동일한 원리이지만, 클릭 몇 번으로 완성됩니다.

진입 방법
상단 툴바 → 특수기능 버튼 클릭 → 패널 상단 탭에서 📥 데이터 입력 (VLOOKUP+) 선택
① 특수기능 클릭
→
② 📥 데이터 입력 (VLOOKUP+) 탭
→
③ 설정 후 실행
사용 순서
아래 5단계를 순서대로 진행하면 됩니다.
단계작업설명
1 왼쪽 파일 불러오기 값을 채워넣을 기준 파일. 현재 열린 파일이 자동으로 로드됩니다. 📁 버튼으로 교체 가능.
2 오른쪽 파일 불러오기 값을 가져올 참조 파일. 📁 버튼 클릭 또는 파일을 끌어다 놓습니다.
3 키 열 선택 양쪽 파일에서 공통으로 사용할 기준 열을 각각 클릭해 지정합니다. (예: 이름, 주민번호) ✨ 자동매칭 버튼으로 헤더 유사도 기반 자동 연결도 가능.
4 복사할 열 선택 오른쪽에서 가져올 열(보라색 영역)과 왼쪽에서 채워넣을 열(보라색 영역)을 각각 선택. + 새 열 추가로 왼쪽 파일에 없는 열도 새로 만들 수 있습니다.
5 실행 · 저장 ▶ 데이터 입력 실행 클릭 → 매칭 결과 확인 후 💾 저장 또는 ⬇ 내보내기
